## Run Locally

Clone the project

```bash
  git clone https://github.com/bhagyamudgal/chimps-academy-nft-collection-app.git
```

Go to the project directory

```bash
  cd chimps-academy-nft-collection-app
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```
